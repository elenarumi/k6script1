# k6script1
Corremos el script con el comando $k6 run blazedemo-test.js en el terminal.

Tambien podemos sobreescribir el número de vus y duración con el comando $k6 run --vus 30 --duration 3m blazedemo-test.js
Y correr nuestro script en la nube $k6 run cloud blazedemo-test.js
